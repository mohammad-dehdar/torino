## torino

An innovative online platform built with Next.js 14 and Tailwind CSS, enabling users to browse and book travel tours in real-time with ease, offering a seamless and responsive experience for modern travelers

## Features

Responsive Design: Ensures a seamless experience across all devices.
User Authentication: Secure sign-up and login with authentication functionality.
SSR & SSG Pages: Combines server-side rendering and static site generation for optimized performance and SEO.
Live Preview: Real-time tour previews for an enhanced booking experience.

## instalation

First,Install torino with npm

```bash
  git clone https://github.com/mohammad-dehdar/torino.git
  cd torino
  npm install
```

run the development server:

```bash
npm run dev

```

To build the project for production:

```bash
npm run build

```
